# 使用 RASA NLU 来构建中文自然语言理解系统（NLU）

本仓库提供前沿、详细和完备的中文自然语言理解系统构建指南。

## 特性
- 提供中文语料库
- 提供语料库转换工具，帮助用户转移语料数据
- 提供多种基于 RASA NLU 的中文语言处理流程
- 提供模型性能评测工具，帮助自动选择和优化模型

## 系统要求

Python 3 (也许支持 python2, 但未经过良好测试)
